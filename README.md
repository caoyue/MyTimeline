MyTimeline
==========

- 聚合分散在各个网站上的 Timeline

    Support:
    - Rss Feed
    - Twitter Oauth
    - Weibo Oauth

- 同步发送消息到第三方

    Support:
    - Twitter
    - Weibo

- 按日期查看过往记录

- 对所有记录的统计图表

    - 按时段统计
    - 按月份统计
    - 按分类统计
